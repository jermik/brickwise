# SEO Terrain UX Philosophy

## Core principles
1. The audit report is the product. Everything else funnels to it.
2. Reports must be readable in under five minutes by a non-technical owner.
3. Findings are ranked by impact, not alphabetized.
4. Every finding includes a "do this this week" action.
5. Lead routing is calm: one form, one CTA, one clear next step.

## Visual rules
- Trust-friendly aesthetic: clean type, plain charts, no neon
- No stock-photo "business owner shaking hands" imagery
- Charts in muted color, labeled axes, no 3D
- Mobile-first because owners read on phones

## Interaction rules
- One primary CTA per page
- Audit form: minimal fields (URL + email is the floor)
- No exit-intent pop-ups
- No chatbots that pretend to be human

## What we do not do
- Gamify SEO with "scores out of 100" that mean nothing
- Hide top findings behind email gates
- Add features that exist to fill a feature comparison table
- Build social/community features
