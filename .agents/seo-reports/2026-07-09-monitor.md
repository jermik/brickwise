## Sitemap & Robots

**Sitemap source** (`app/sitemap.ts`): Covers static routes, rankings, city pages, property pages, market reports, Algorand project pages. Uses `new Date()` for `lastModified` on static routes (always today). Property pages use `new Date(p.lastUpdated)` correctly.
- ⚠️ POTENTIAL: Static routes using `new Date()` for `lastModified` may signal to Googlebot that all static pages are updated daily, which is inaccurate for low-change pages like `/privacy`, `/about`, `/contact`. Not harmful, but mildly misleading. Whitelist-safe fix.

**Robots.txt source** (`app/robots.ts`): Correctly disallows `/dashboard`, `/portfolio`, `/watchlist`, `/settings`, `/sign-in`, `/sign-up`, `/announce`, `/seo-dashboard`, `/crm`, `/growthos`, `/api/`. Sitemap reference correct (`https://brickwise.pro/sitemap.xml`). Host set correctly.
- ⚠️ `/learn/` is NOT in the disallow list — correctly crawlable.
- ✓ No public-facing revenue/content paths blocked.
