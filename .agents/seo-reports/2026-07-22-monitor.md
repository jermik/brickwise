## Out of Scope — Needs Human

1. **`/analyzer` missing metadata** — The page is `"use client"` and cannot export `metadata` at the module level (Next.js constraint). Fix requires either: (a) adding a `layout.tsx` inside `app/analyzer/` that exports metadata and keeps the page as a client component, or (b) converting the page to a server component with a client-component island. Both options touch the component architecture. Not a whitelist-safe edit.

2. **`FallbackImg` CLS risk** — The `FallbackImg` component in `components/ui/` is on the blacklist. Fixing requires adding a `width`/`height` or using Next.js `<Image>` in that component. Owner should evaluate whether per-callsite `width`/`height` props or a component refactor is preferred.

3. **General Sans font self-hosting** — Fontshare CDN dependency is outside the whitelist to modify (`next.config.*`, `globals.css` if that's where it's referenced). Owner should evaluate self-hosting to guarantee `font-display: swap` and eliminate the external CDN dependency.

4. **`lastmod: new Date()` on all static sitemap routes** — Every sitemap refresh reports today as the lastmod for all static pages, regardless of whether content changed. This inflates Googlebot crawl budget. Would require explicit static date constants per route — outside daily routine scope.
