## Passing checks

- **HTML lang**: `lang="en"` on root `<html>` ✅
- **Exactly one `<h1>`**: Verified on all 27 pages ✅
- **Canonical links**: All pages except `/analyzer` set a self-referential canonical ✅
- **OG tags**: All PublicShell pages have og:title + og:description (at minimum via layout inheritance) ✅
- **OG image**: Layout global `/opengraph-image` (1200×630) is inherited by all pages; compare/realt-vs-lofty, rankings, and property pages override with page-specific images ✅
- **JSON-LD**: Layout emits global Organization + WebSite schema. City pages: Dataset. Learn pages: Article + FAQPage. Property pages: inherited OG. All JSON-LD uses `JSON.stringify()` within server components — parse-safe ✅
- **Font display: swap**: All Next.js fonts (`Inter`, `DM Mono`, `DM Serif Display`) use `display: "swap"`. External `General Sans` from Fontshare loads non-blocking via `NonBlockingStylesheet` pattern with `display=swap` in URL ✅
- **Robots.txt**: Correctly disallows all private routes (`/dashboard`, `/portfolio`, `/watchlist`, `/settings`, `/sign-in`, `/sign-up`, `/announce`, `/seo-dashboard`, `/crm`, `/growthos`, `/api/`). Allows all public routes. Sitemap directive points to `https://brickwise.pro/sitemap.xml` ✅
- **Sitemap structure**: Covers static routes, property routes (all 479), city routes (deduplicated), rankings categories, market reports, algorand projects. Priority hierarchy is logical ✅
- **Independence statement**: Not weakened in any page reviewed ✅
- **Anti-pattern compliance**: No hype language, no affiliation claims, no AI buzzwords observed in metadata copy ✅
