## Infrastructure Note

**brickwise.pro returned HTTP 403 for all URL checks** (homepage, sitemap.xml, robots.txt, all page URLs). This prevents live-render verification of title tags, canonical links, JSON-LD rendering, and sitemap reachability. Likely cause: Cloudflare bot protection blocking the audit agent's user-agent.

**Impact:** This routine cannot verify whether server-rendered HTML actually contains the metadata defined in source. It can only verify the source definitions. If the site is deployed, the 403 is itself an SEO concern — Googlebot may have similar issues unless Cloudflare is configured to allow its crawler UA.

Recommend: Confirm Google Search Console shows no crawl errors. If Cloudflare is blocking, verify that "Known Bots" → Googlebot is set to Allow in the WAF rules.
