## Out of Scope — Needs Human

### `/analyzer/page.tsx` — Missing metadata (client component)
`app/analyzer/page.tsx` is a `"use client"` component. Next.js App Router does not allow metadata exports from client components. As a result, Googlebot sees no `<title>` or `<meta description>` for this page. The fix requires splitting the interactive UI into a child client component and keeping the page.tsx as a server component that exports metadata — a structural refactor outside the SEO whitelist.

**Recommended fix**: Create `app/analyzer/_client.tsx` (move all client logic there), rewrite `page.tsx` as a server component with a metadata export that wraps `<_client />`.
